Anime Verse is a mobile application designed to provide a seamless experience for browsing and reviewing anime movies. This project showcases practical implementation of modern mobile development principles, focusing on clean architecture, responsive UI, and intuitive user interaction.

Built as part of an academic mobile programming project, Anime Verse emphasizes both technical quality and user-centered design.

Project Overview

Anime Verse allows users to:

Discover anime movies through a simple and elegant interface

View detailed information including synopsis and ratings

Submit personal reviews and ratings

Navigate smoothly between views using a structured navigation flow

The application was developed with a focus on scalability, maintainability, and clean code practices.
