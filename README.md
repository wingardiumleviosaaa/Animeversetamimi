Anime Verse is a mobile application designed to provide a seamless experience for browsing and reviewing anime movies. This project showcases practical implementation of modern mobile development principles, focusing on clean architecture, responsive UI, and intuitive user interaction. Built as part of an academic mobile programming project, Anime Verse emphasizes both technical quality and user-centered design.


Project Overview:
1. Anime Verse allows users to:
- Discover anime movies through a simple and elegant interface
- View detailed information including synopsis and ratings
- Add personal favourite onto a dedicated favorite page to save the tittle
- Navigate smoothly between views using a structured navigation flow

2. The application was developed with a focus on scalability, maintainability, and clean code practices.

3. Key Highlights:
- Modern and minimal UI design
- Structured project architecture
- Smooth navigation and user experience
- Beginner-friendly and well-documented codebase

4. Technology Stack
- Language: Kotlin / Java / Dart
- Framework: Android Native / Flutter

5. IDE: Android Studio
